NOAVIA — Data Privacy & GDPR Policy
Data Processing
NOAVIA processes all client data exclusively within the European Union. Our infrastructure is hosted on German data centers operated by Hetzner Online GmbH in Falkenstein and Nuremberg, Germany. All data centers are ISO 27001 certified.

Your Rights Under GDPR
As a data subject, you have the following rights:
Auskunftsrecht (Art. 15 DSGVO): Request a copy of all personal data we hold about you or your company.
Recht auf Berichtigung (Art. 16 DSGVO): Request correction of inaccurate data.
Recht auf Löschung (Art. 17 DSGVO): Request deletion of your data ("Recht auf Vergessenwerden").
Recht auf Datenübertragbarkeit (Art. 20 DSGVO): Receive your data in machine-readable format (JSON, CSV).
Widerspruchsrecht (Art. 21 DSGVO): Object to processing for specific purposes.
To exercise any of these rights, contact datenschutz@noavia.de with your Kundennummer. We respond within 30 days.
Data Retention
Active client data (AI models, workflow configs, processed documents): Retained as long as the contract is active.
After contract end: Data retained for 90 days for transition, then permanently deleted.
Billing records: Retained for 10 years as required by German commercial law (§257 HGB, §147 AO).
Support tickets: Retained for 3 years after resolution, then anonymized.
AI-Specific Data Handling
Client documents used for knowledge base training are stored only in the client's dedicated Qdrant vector collection — never shared across clients.
AI agent interactions (queries and responses) are logged for quality assurance. Logs are retained for 30 days unless longer retention is requested.
NOAVIA does not use client data to train general AI models. Your data is used exclusively for your own agents.
When using OpenAI APIs, data is processed through the EU-hosted endpoint with a Data Processing Agreement (DPA) in place. No client data is used for OpenAI model training (API usage is excluded per OpenAI's enterprise terms).
Sub-Processors
We use the following sub-processors:
Hetzner Online GmbH (infrastructure hosting, Falkenstein/Nuremberg, Germany)
OpenAI Inc. (AI inference, EU API endpoint, DPA in place)
Stripe Inc. (payment processing, EU data center, Dublin)
Hetzner Mail (transactional emails, Germany)
Security Measures
AES-256 encryption for data at rest
TLS 1.3 encryption for data in transit
Client data isolation: Each client has a dedicated vector database collection and workflow namespace
Role-based access control (RBAC) for all NOAVIA staff
Regular penetration testing (quarterly, conducted by an independent German security firm)
Incident response plan: 72-hour breach notification as required by Art. 33 DSGVO
Annual security awareness training for all employees
